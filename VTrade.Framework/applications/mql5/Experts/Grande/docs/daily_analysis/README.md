# Daily Analysis Reports

This directory contains analysis reports generated by the Grande Trading System optimization pipeline.

## Report Types

### Daily Analysis Reports
- **Filename**: `DAILY_ANALYSIS_REPORT_YYYYMMDD.md`
- **Generated by**: `.\scripts\RunDailyAnalysis.ps1`
- **Frequency**: Run daily or after significant trading activity
- **Content**:
  - Signal type performance (win rates, profit/loss)
  - Regime-specific analysis
  - Optimal RSI/ADX thresholds
  - Rejection reason breakdown
  - EA parameter recommendations with confidence levels

### Optimization Findings
- **Filename**: `OPTIMIZATION_FINDINGS_YYYYMMDD.md`
- **Generated by**: Manual analysis or automated reports
- **Content**:
  - Critical issues requiring immediate attention
  - Performance analysis of closed trades
  - Risk management recommendations
  - Specific parameter adjustments with rationale

## Usage

### Generate New Report
```powershell
# From workspace root
.\scripts\RunDailyAnalysis.ps1
```

### Read Latest Report
```powershell
# Find most recent
Get-ChildItem docs\daily_analysis\DAILY_ANALYSIS_REPORT_*.md | Sort-Object LastWriteTime -Descending | Select-Object -First 1

# Or open in editor
code docs\daily_analysis\DAILY_ANALYSIS_REPORT_$(Get-Date -Format "yyyyMMdd").md
```

## Report Structure

Reports typically include:

1. **Executive Summary**: Overall performance metrics
2. **Critical Findings**: Issues requiring immediate action
3. **Performance Analysis**: By signal type, direction, symbol
4. **Optimization Recommendations**: Parameter changes with confidence levels
5. **Action Plan**: Immediate, weekly, and monthly tasks

## Statistical Confidence

Recommendations are marked with confidence levels:
- **HIGH**: 20+ trades, clear pattern, apply immediately
- **MEDIUM**: 10-19 trades, emerging pattern, test in Strategy Tester
- **LOW**: <10 trades, insufficient data, wait for more trades
